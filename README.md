## Requirements:

node.js, npm


## Getting started:

1. git clone project

2. In terminal or command prompt move to folder fcg-frontend-component

3. Checkout development branch `git checkout development`

3. `npm install`

4. `npm start`

5. Web browser opens on localhost:3000



## Build for produciton:

`npm run build`

Testing:

`npm test`
